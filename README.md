# Rime 万象更新工具

## 项目简介

本工具用于自动更新Rime输入法的万象方案、词库和模型，支持Windows、macOS和Linux系统。

## 系统版本

- [x] [Windows版本](./Windows/README.md)
  - [x] [PowerShell版本](./Windows/PowerShell/README.md)
  - [x] [Python版本](./Windows/Python/README.md)
- [ ] macOS版本
- [ ] Linux版本

## 使用说明

1. 选择对应的系统版本
2. 按照说明文档进行操作
3. 工具将自动完成更新
